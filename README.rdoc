== README

Mock Rotten Tomatoes web app created with Rails as Lighthouse Labs course work (week 5).

Implemented ActionMailer to email users when deleted by admin.
Users can add a movie, upload photo, and search.
Implemented Admin accounts.
Implemented pagination with Kaminari gem.

Created with 'rails', '4.0.0'.

Database: sqlite3 as the database for Active Record.

Password encryption 'bcrypt' gem.
